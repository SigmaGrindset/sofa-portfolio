# sofa-portfolio

Personal portfolio — Antonio Batarilović.

## Stack
- Vite + React 18 + TypeScript
- TanStack Router
- Tailwind CSS v4
- GSAP (ScrollTrigger)

## Develop
```bash
npm install
npm run dev
```

## Deploy
Vercel (framework preset: Vite). `vercel.json` rewrites all routes to `/` for client-side routing.
